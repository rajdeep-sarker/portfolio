1. **profile.jpg** - Your profile picture
   - Recommended size: 300x300px
   - Format: JPG or PNG
   - Should be a clear, professional photo

2. **project1.jpg** - First project screenshot
   - Recommended size: 600x400px
   - Format: JPG or PNG

3. **project2.jpg** - Second project screenshot
   - Recommended size: 600x400px
   - Format: JPG or PNG

4. **project3.jpg** - Third project screenshot

## Tips for Images

- Use optimized images to ensure fast loading times
- Maintain consistent aspect ratios for project images
- Use high-quality images but keep file sizes under 500KB
- Consider using tools like TinyPNG or ImageOptim to compress images
