# My Portfolio

A modern, responsive portfolio website for HSC Science Class XI students.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and professional design with smooth animations
- **Interactive Elements**: Animated skill bars, hover effects, and smooth scrolling
- **Multiple Sections**:
  - Hero section with introduction
  - About section with personal information
  - Education timeline
  - Skills showcase (academic, technical, and soft skills)
  - Projects gallery
  - Achievements section
  - Contact form with social links

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (Vanilla JS)
- Font Awesome Icons

## Getting Started

1. Clone or download this repository
2. Replace placeholder content with your own information:
   - Update name, email, and contact details
   - Add your profile picture to `assets/profile.jpg`
   - Add project images to `assets/` folder
   - Modify education, skills, and achievements sections
   - Update social media links

3. Open `index.html` in your browser to view the portfolio

## Customization

### Colors
You can customize the color scheme by modifying CSS variables in `css/style.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #7c3aed;
    --dark-bg: #0f172a;
    --light-bg: #f8fafc;
}
```

### Content
Edit the following files:
- `index.html` - Update all text content and links
- `css/style.css` - Modify styles and layout
- `js/script.js` - Customize interactive features

### Images
Place your images in the `assets/` folder:
- `profile.jpg` - Your profile picture (recommended: 300x300px)
- `project1.jpg`, `project2.jpg`, `project3.jpg` - Project screenshots

## Tips for Students

1. **Personalize your content**: Make sure all information reflects your actual achievements and skills
2. **Add real projects**: Replace placeholder projects with your actual work
3. **Keep it updated**: Regularly update your portfolio as you gain new skills and complete projects
4. **Optimize images**: Use compressed images for faster loading
5. **Test on mobile**: Ensure everything works well on mobile devices
